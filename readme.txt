/--------- How to play --------/
If running both clients and server on one machine:
    1. Run 'python Server.py' in the terminal.
    2. Run 'python Client.py' in two different terminals for each client.
    3. Play the game.
Otherwise:
    1. Run 'python Server.py 127.0.0.1'
        or 'python Server.py 127.0.0.1 80', replacing 127.0.0.1 and 80 with any other IP address and port number.
    2. Run 'python Client.py 127.0.0.1'
        or 'python Client.py 127.0.0.1 80', replacing 127.0.0.1 and 80 with any other IP address and port number.
    3. Play the game.
